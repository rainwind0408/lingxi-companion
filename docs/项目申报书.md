## 二、立项依据

### 2.1 研究背景

随着在线教育的高速发展，网络边缘场景下的学习者面临严峻挑战。偏远地区学习者、设备受限用户以及网络不稳定环境中的学习者，往往因为传统平台的高带宽需求而无法获得流畅的学习体验。

与此同时，情感计算在教育领域的价值日益凸显。实时感知学习者的情绪状态，能够帮助平台动态调整内容呈现方式，提高学习效率与学习者满意度。

### 2.2 核心问题

1. **边缘场景下的计算资源受限**：传统情感识别模型体积庞大，无法在边缘设备上高效运行
2. **多模态感知的协同难题**：文本、语音、表情等多种信号如何高效融合
3. **带宽自适应内容推送**：网络状态动态变化下如何保证内容质量
4. **轻量化与准确率的平衡**：模型压缩带来的精度损失问题

### 2.3 研究思路

本项目借鉴生物群体的协同感知机制（如蜂群、蚁群），设计多智能体分布式感知框架。核心思路如下：

- **感知层**：部署多个轻量级感知 Agent，分别处理文本、语音等单一模态信号
- **融合层**：通过协同注意力机制整合各 Agent 的输出
- **决策层**：基于融合结果动态调整内容推送策略
- **反馈层**：建立长短期情感状态模型，形成个性化学习闭环

### 2.4 创新点

1. **仿生架构**：首次将生物群体协同感知机制引入教育情感计算领域
2. **轻量化设计**：采用知识蒸馏与量化压缩技术，确保模型可在边缘设备运行
3. **协同感知**：多 Agent 并行处理，通过协同注意力实现信息高效融合
4. **带宽自适应**：结合内容预加载与质量动态调整，保障学习体验
