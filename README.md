# 灵犀学伴 (Lingxi Companion)

> 面向在线学习边缘场景的仿生多智能体轻量情感协同感知系统
> 
> 江苏省大学生创新创业训练计划项目

---

## 📋 项目简介

本项目旨在为在线学习中处于网络边缘、带宽受限或设备老旧的学习者，提供一套基于仿生多智能体的轻量化情感协同感知系统。通过模拟生物群体的协同感知机制，系统能够在资源极度受限的环境下，实时检测学习者的情绪状态，并动态调整内容推送策略，提升学习体验与效果。

## 🎯 项目目标

- 构建基于多智能体协同的边缘感知计算架构
- 设计轻量化情感识别模型（可离线运行）
- 实现带宽自适应内容推送机制
- 研发"灵犀"核心引擎与配套应用套件

## 🔧 技术方案

### 系统架构

```
┌─────────────────────────────────────────────────────┐
│                    用户端应用层                      │
│  (Web / Mobile / Desktop)                           │
└─────────────────────┬─────────────────────────────┘
                      │
┌─────────────────────▼─────────────────────────────┐
│                  API 网关层                         │
│  (负载均衡 / 限流 / 缓存)                           │
└─────────────────────┬─────────────────────────────┘
                      │
┌─────────────────────▼─────────────────────────────┐
│               多智能体协同引擎 (灵犀核心)              │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐    │
│  │ 感知Agent  │ │ 决策Agent   │ │ 执行Agent   │    │
│  └─────────────┘ └─────────────┘ └─────────────┘    │
└─────────────────────┬─────────────────────────────┘
                      │
┌─────────────────────▼─────────────────────────────┐
│           边缘计算节点层 (可选本地部署)               │
│  (轻量化模型 / 离线推理 / 缓存)                     │
└─────────────────────────────────────────────────────┘
```

### 核心技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端 | Node.js / Python |
| 数据库 | SQLite / Redis |
| 前端 | React / Vue |
| 模型 | TensorFlow Lite / ONNX |
| 通信 | WebSocket / HTTP |

### 核心模块

1. **感知模块** - 采集用户行为数据（点击、停留、输入节奏等）
2. **分析模块** - 本地轻量化情感分析（离线可用）
3. **决策模块** - 多智能体协同推理
4. **推送模块** - 自适应内容策略
