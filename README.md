# 灵犀学伴 (Lingxi Companion)

> 面向在线学习边缘场景的仿生多智能体轻量情感协同感知系统
> 
> 江苏省大学生创新创业训练计划项目

---

## 📋 项目简介

本项目旨在为在线学习中处于网络边缘、带宽受限或设备老旧的学习者，提供一套基于仿生多智能体的轻量化情感协同感知系统。通过模拟生物群体的协同感知机制，系统能够在资源极度受限的环境下，实时检测学习者的情绪状态，并动态调整内容推送策略，提升学习体验与效果。

## 🎯 项目目标

- 构建基于多智能体协同的边缘感知计算架构
- 设计轻量化情感识别模型（可离线运行）
- 实现带宽自适应内容推送机制
- 研发"灵犀"核心引擎与配套应用套件
